# Code of Conduct

## Our Pledge
We promise to make this project welcoming and harassment-free for everyone.

## Expected Behavior
- Be respectful.
- Be helpful.
- Accept feedback politely.
- Focus on the issue, not the person.

## Unacceptable Behavior
- Harassment or personal attacks.
- Disrespectful or rude comments.
- Sharing sensitive information.

## Reporting Issues
If you see any bad behavior:
- Open an issue in the repository  
OR  
- Contact the maintainers directly.

Thank you for helping us keep this project healthy and friendly!
